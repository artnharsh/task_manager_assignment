# Personal Task Tracker

## 📖 Description

- Simple task manager
- User Friendly UI
- Task sorting/filtering feature
- Add, edit, delete and update task feature


## ️ Setup Instructions

1. Clone the repository

2. Run `npm install`

3. Run `npm run dev`

4. Open [http://localhost:5173](http://localhost:5173)

## 🧰 Technologies Used

- React.js
